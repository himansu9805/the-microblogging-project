# The Microblogging Project

This project was bootstrapped by [Himansu](https://himansu.dev) with an aim to develop an indigenous and creative microblogging platform for Indian people.

## UI designs and plans

For designing UI, we are using Figma.

Go to this [link](https://www.figma.com/file/Y6PlrsaTC5ufy7dohBdAnl/Microblogging-Platform-Designs?node-id=0%3A1) to see the designs and plans.

## Technologies used

The front end of this project
1. React
2. Custom CSS written using SASS

The backend of this project
1. Spring Boot (Yet to be implemented)
2. PostgreSQL

For deployment we are planning to use Microsoft Azure's services.
